# resume-generator

A project born from the desire to overengineer what could have been a Google Doc.

`resume-generator` turns resume bullet points into a professional-looking PDF.

## Quick Start

Install deps
```
brew install pandoc mactex
```

Edit resume
```
nano source.md
```

Generate PDF file
```
make
```
